# product_auto_pricing (Odoo 16)

Addon Odoo 16 pour calcul automatique du pricing (règles internes).

## Installation
- Copier le dossier `product_auto_pricing` dans un chemin présent dans `addons_path`
- Redémarrer Odoo
- Apps > Update Apps List
- Installer le module

## Notes
- Dépendances: voir `__manifest__.py`
